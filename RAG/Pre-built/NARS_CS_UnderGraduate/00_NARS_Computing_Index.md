# Index: NARS for Computing and Information — 1st Edition, October 2010
## National Authority for Quality Assurance and Accreditation of Education (NAQAAE)

---

## Overview

This document defines the **National Academic Reference Standards (NARS)** for the Computing and Information sector in Egypt. It sets the minimum expected knowledge, skills and competencies for graduates of undergraduate computing programs.

**Key Definitions:**
- **NARS**: Reference points by NAQAAE describing the minimum knowledge and skills for a program of study
- **Academic Standards**: Institution-defined standards that must *surpass* NARS and be approved by NAQAAE
- **ILOs (Intended Learning Outcomes)**: What learners should be able to do after completing an activity
- **Graduate Attributes**: Competencies expected from the graduate upon program completion
